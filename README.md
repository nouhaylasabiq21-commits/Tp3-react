# TP3 React – Navigation et Médias

## 📌 Objectif

Réaliser une application React avec navigation entre pages, intégration de médias (image, audio) et création de composants.

## 🧩 Fonctionnalités

* Navigation avec **React Router**
* Page **Accueil**, **À propos** et **Contact**
* Affichage d’un **logo**
* Lecture d’un **audio**
* Liste de tâches
* Rendu conditionnel (notifications)
* Liste dynamique avec `map()`
* Galerie d’images

## 📂 Structure principale

* `App.js` : gestion des routes
* `Accueil.js` : page principale
* `Apropos.js`, `Contact.js` : autres pages
* Composants : `Logo`, `LecteurAudio`, `ListeTaches`, `Galerie`, etc.
